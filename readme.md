## Getting started

Clone the repository:

```shell
git clone https://github.com/RyanNieuwoudt/ParseDontValidate-csharp.git
```

Change to the project directory:

```shell
cd ParseDontValidate-csharp
```

Run the sample:

```shell
dotnet run
```
